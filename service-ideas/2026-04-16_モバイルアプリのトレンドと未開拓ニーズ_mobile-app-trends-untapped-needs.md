## 市場概要

2025〜2026年のモバイルアプリ市場は以下のマクロトレンドが交差している。

| トレンド | 概要 |
|---|---|
| AIネイティブ化 | ChatGPT・Perplexity・Claudeなど、AIを中核に置くアプリが急増 |
| ウェルネス・メンタルヘルス | 睡眠・瞑想・ストレス管理への需要が継続拡大 |
| Gen Z向けゲーミフィケーション | 習慣化・学習系アプリのゲーム的UXが当たり前に |
| プライバシーファースト | ローカル処理・最小データ収集を売りにするアプリが増加 |
| スーパーアプリ疲れへの反動 | 単機能・シンプルアプリへの回帰ニーズ |

日本国内ではスマホ普及率が高く（約90%超）、シニア層のスマホ移行も進んでいる。一方で「日本語 × ローカルニーズ」に特化したアプリは海外アプリに勝てる余地が大きい。

## 具体的なサービスアイデア

---

### アイデア1: AIパーソナル家計コーチ「MoneyMentor」

**コンセプト**  
銀行口座・クレジットカードの明細を自動取得し、AIが支出パターンを分析して具体的な節約アドバイスをプッシュ通知で届けるアプリ。単なる「記録ツール」を超え、毎日ひとこと話しかけてくる"お金のコーチ"を目指す。

> 「先月カフェに9,800円使いました。週2日だけ自炊に変えると月4,000円浮きます。今週試してみますか?」

**想定ユーザー**
- 20〜35歳の会社員・フリーランス
- 家計簿アプリを入れても3日で挫折する層
- 「なんとなくお金が貯まらない」と感じているが原因が分からない人

**収益モデル**
- フリーミアム: 基本機能無料、AI分析・アドバイスはプレミアム月額980円
- 年払い割引（8,800円/年）で継続率向上
- 金融商品との提携アフィリエイト（節約目標達成後に投資信託を提案など）

**技術スタック**
- フロントエンド: Flutter（iOS/Android同時対応）
- バックエンド: FastAPI + Neon PostgreSQL
- AI: Claude API（支出分析・アドバイス生成）
- 口座連携: マネーフォワードAPI or Moneytree LINK
- 通知: Firebase Cloud Messaging

**差別化ポイント**
- 既存家計簿アプリ（マネーフォワード・Zaim）は「記録」が主軸。MoneyMentorは「行動変容の促進」に特化
- プッシュ通知でAIコーチが毎日声がけ → 習慣化ループを形成
- 目標設定（旅行・貯金）に向けた逆算プランを自動生成

**個人開発者が1人で作れるか: ★★★★☆**  
Claude APIと既存の口座連携SDKを組み合わせれば、AI部分の実装コストは低い。マネーフォワードAPIの審査が必要だが、個人開発者でも申請可能。MVP（口座連携なし・手入力版）なら2〜3週間で完成できる。

### アイデア2: 職人・クリエイター向けスキル成長記録「CraftLog」

**コンセプト**  
料理人・大工・陶芸家・美容師など「手に職」系の職業に向けた、技術習得の軌跡を記録するアプリ。写真・動画・メモで作品と学びを蓄積し、師匠からのフィードバックを受け取れるプラットフォーム。

**想定ユーザー**
- 職人見習い・修行中の若者（18〜30歳）
- 副業・趣味でクラフト系を極めたい社会人
- 弟子の成長を管理したい師匠・講師

**収益モデル**
- 個人プラン: 月額980円（無制限記録・AI振り返り機能）
- 道場・スクールプラン: 月額9,800円〜（師匠1名 + 弟子10名まで）
- 認定バッジ機能: 師匠が発行する技術証明書をデジタル化 → 転職・副業時のポートフォリオとして活用

**技術スタック**
- フロントエンド: React Native
- バックエンド: FastAPI + Neon PostgreSQL
- ストレージ: Cloudflare R2（動画・画像）
- AI: Claude API（記録から学習ポイントを抽出・振り返りレポート生成）

**差別化ポイント**
- 一般的なSNSや日記アプリと違い「技術の成長曲線」が可視化される
- 師弟関係をデジタルで管理できる機能は他にない
- 伝統工芸・職人文化に強い日本市場との親和性が高い

**個人開発者が1人で作れるか: ★★★☆☆**  
コア機能（記録・フィードバック）は1人でも十分実装可能。動画のエンコード処理に工数がかかるが、Cloudflare Streamを使えば削減できる。初期ユーザー獲得のために職人コミュニティへのリアル営業が必要になる点がボトルネック。

### アイデア3: 近所の「今日だけ」お得情報通知アプリ「キョウダケ」

**コンセプト**  
スーパー・惣菜屋・パン屋などの近隣店舗が「今日の訳あり品・残り物・限定セール」を投稿し、近所のユーザーにリアルタイム通知するアプリ。食品ロス削減とお得購買を同時に実現する。

**想定ユーザー**
- 節約志向の主婦・一人暮らし社会人
- 食品ロス削減に関心があるエコ志向ユーザー
- 地元の小売店・惣菜店（掲載する側）

**収益モデル**
- 店舗向け掲載料: 月額3,000〜8,000円（掲載件数・エリア範囲で段階設定）
- ユーザー向けプレミアム: 月額380円（通知エリア拡大・カテゴリフィルター強化）
- 成功報酬型オプション: クーポン利用数に応じた従量課金

**技術スタック**
- フロントエンド: Flutter
- バックエンド: FastAPI + Supabase（リアルタイム通知に強い）
- 位置情報: Google Maps Platform
- プッシュ通知: Firebase Cloud Messaging

**差別化ポイント**
- チラシアプリ（Shufoo!等）は「週間広告」が主流。キョウダケは「今日の今」に特化したリアルタイム性が武器
- 食品ロス削減という社会的意義でメディア露出を狙いやすい
- 小規模店舗こそ恩恵が大きく、チラシ印刷コストの代替として訴求可能

**個人開発者が1人で作れるか: ★★☆☆☆**  
技術的な実装は個人でも可能だが、店舗開拓というオフライン営業が不可欠。地域限定でスモールスタートし、成功事例を作ってから横展開する戦略が現実的。エリアを1市区町村に絞ったMVPなら半年で検証可能。

### アイデア4: センサーだけで眠りを測る「NeruAI」

**コンセプト**  
スマートウォッチ不要。スマホのマイク・加速度センサーだけで睡眠を計測し、AIが「眠りの質」と「翌日のコンディション」の相関を分析するアプリ。「なぜ今日はだるいのか」を睡眠データから説明する。

**想定ユーザー**
- 睡眠の質に悩む30〜50代のビジネスパーソン
- ウェアラブル端末を持っていないが睡眠を改善したい人
- 残業・不規則生活で睡眠負債を抱えているエンジニア・医療従事者

**収益モデル**
- 基本無料（7日間の履歴）+ プレミアム月額480円（90日分析・AI改善提案）
- 寝具ブランド・サプリメーカーとのタイアップ広告
- 法人向けウェルネスプログラム（従業員の睡眠スコア集計）月額50,000円〜

**技術スタック**
- iOS: Swift + HealthKit（Apple Watch連携も可能にして上位互換に）
- Android: Kotlin + Health Connect
- バックエンド: FastAPI + Neon PostgreSQL
- AI/ML: 睡眠ステージ推定モデル（TensorFlow Lite でオンデバイス処理）+ Claude APIで改善アドバイス生成

**差別化ポイント**
- 競合（Sleep Cycle等）もセンサー計測だが、AIによる「翌日コンディション予測」機能は希少
- オンデバイスML処理でプライバシー懸念を排除
- 「睡眠負債の返済スケジュール」自動生成機能で差別化

**個人開発者が1人で作れるか: ★★★☆☆**  
センサーからの睡眠ステージ推定はMLモデルの精度が肝。既存のオープンソースモデルを転用すれば個人でも実装可能だが、精度担保のために実データでのファインチューニングが必要。AI分析はClaude APIに任せることで開発工数を抑えられる。

## まとめ・個人開発者へのアドバイス

- **最短MVPを狙うなら MoneyMentor**: 口座連携を後回しにして「手入力 + AIアドバイス」から始めれば2〜3週間でApp Store申請できる。Claude APIの利用料も月数千円規模から始められる。
- **差別化しやすいのは CraftLog**: 職人向けSaaSは競合が少なく、1,000人の熱狂的ユーザーを先に獲得する戦略が機能しやすい。
- **NeruAI は技術的チャレンジ好き向け**: MLモデルの精度が商品の質に直結するため、機械学習の知識があるエンジニアには面白い挑戦になる。
- **キョウダケはチームで**: オフライン営業コンポーネントがあるため、営業得意な共同創業者がいると一気に加速する。

モバイルアプリ市場は「ネタ切れ」ではなく、AIとリアルタイム性の掛け合わせで新たな価値提供の余地が十分残っている。「記録するだけ」から「行動を変える」へのシフトが次の一手。
